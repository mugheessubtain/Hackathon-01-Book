# ہفتہ 1: ROS 2 کا تعارف

ROS 2 میں خوش آمدید! اس باب میں، آپ روبوٹ آپریٹنگ سسٹم 2 کی بنیادیں سیکھیں گے۔

## ROS 2 کیا ہے؟

ROS 2 (روبوٹ آپریٹنگ سسٹم 2) روبوٹکس کے لیے ایک اوپن سورس مڈل ویئر فریم ورک ہے۔ یہ فراہم کرتا ہے:
- تقسیم شدہ سسٹمز کے لیے **کمیونیکیشن انفراسٹرکچر**
- روبوٹ ایپلیکیشنز بنانے کے لیے **ٹولز اور لائبریریز**
- مختلف سینسرز اور ایکچویٹرز کے لیے **ہارڈویئر ایبسٹریکشن**

## اہم تصورات

### نوڈز
نوڈز ROS 2 میں بنیادی کمپیوٹیشنل یونٹس ہیں۔ ہر نوڈ ایک پروسیس کی نمائندگی کرتا ہے جو مخصوص کام انجام دیتا ہے۔

```python
import rclpy
from rclpy.node import Node

class MyNode(Node):
    def __init__(self):
        super().__init__('my_node')
        self.get_logger().info('نوڈ شروع ہو گیا!')

def main():
    rclpy.init()
    node = MyNode()
    rclpy.spin(node)
    rclpy.shutdown()
```
