# ہفتہ 9: روبوٹ سمولیشن کے لیے Isaac Sim

ایڈوانسڈ روبوٹ سمولیشن سیناریوز کے لیے Isaac Sim استعمال کرنا سیکھیں۔

## ماحول بنانا

Isaac Sim فراہم کرتا ہے:
- **پہلے سے بنے ہوئے اثاثے** اور روبوٹس
- **فزکس انجنز** (PhysX)
- **سینسر سمولیشن** (RGB، ڈیپتھ، لیڈار)

```python
from omni.isaac.kit import SimulationApp
simulation_app = SimulationApp({"headless": False})

from omni.isaac.core import World
world = World()
world.scene.add_default_ground_plane()
```

## سینسر انٹیگریشن

- RGB کیمرے
- ڈیپتھ کیمرے
- LiDAR سینسرز
- کانٹیکٹ سینسرز

## مشق

کیمرہ سینسرز کے ساتھ Isaac Sim میں روبوٹ سیٹ اپ کریں۔
