# ہفتہ 12: روبوٹ کمانڈز کے لیے لینگویج ماڈلز

دریافت کریں کہ لینگویج ماڈلز روبوٹس کے ساتھ قدرتی زبان میں تعامل کیسے ممکن بناتے ہیں۔

## روبوٹکس کے لیے زبان

لینگویج ماڈلز روبوٹس کو اجازت دیتے ہیں:
- قدرتی زبان کے کمانڈز **سمجھیں**
- ٹاسک سیکوینسز **پلان** کریں
- ایکشنز کے بارے میں **استدلال** کریں

## VLA ماڈلز

ویژن-لینگویج-ایکشن ماڈلز یکجا کرتے ہیں:
- **ویژن**: ماحول دیکھیں
- **لینگویج**: کمانڈز سمجھیں
- **ایکشن**: موٹر کمانڈز ایگزیکیوٹ کریں

```python
# مثال: VLA ماڈل استعمال کرنا
from transformers import AutoModel

vla_model = AutoModel.from_pretrained("openvla/openvla-7b")
```
