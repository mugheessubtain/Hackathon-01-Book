# Semaine 9 : Isaac Sim pour la Simulation de Robots

Apprenez à utiliser Isaac Sim pour des scénarios avancés de simulation de robots.

## Création d'Environnements

Isaac Sim fournit :
- **Assets et robots** pré-construits
- **Moteurs physiques** (PhysX)
- **Simulation de capteurs** (RGB, profondeur, lidar)

```python
from omni.isaac.kit import SimulationApp
simulation_app = SimulationApp({"headless": False})

from omni.isaac.core import World
world = World()
world.scene.add_default_ground_plane()
```

## Intégration des Capteurs

- Caméras RGB
- Caméras de profondeur
- Capteurs LiDAR
- Capteurs de contact

## Exercice

Configurez un robot dans Isaac Sim avec des capteurs de caméra.
