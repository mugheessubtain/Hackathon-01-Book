# Semaine 7 : URDF et Modèles de Robots

Apprenez à décrire des robots en utilisant URDF (Unified Robot Description Format).

## Qu'est-ce que URDF ?

URDF est un format XML pour représenter les modèles de robots, incluant :
- **Links** : Corps rigides
- **Joints** : Connexions entre les links
- **Visual** : Apparence
- **Collision** : Limites physiques

```xml
<robot name="my_robot">
  <link name="base_link">
    <visual>
      <geometry>
        <box size="0.6 0.3 0.2"/>
      </geometry>
    </visual>
  </link>
  
  <joint name="wheel_joint" type="continuous">
    <parent link="base_link"/>
    <child link="wheel_link"/>
    <origin xyz="0.2 0 -0.1"/>
    <axis xyz="0 1 0"/>
  </joint>
</robot>
```
