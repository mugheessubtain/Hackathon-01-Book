# الأسبوع 12: نماذج اللغة لأوامر الروبوت

اكتشف كيف تُمكّن نماذج اللغة التفاعل باللغة الطبيعية مع الروبوتات.

## اللغة للروبوتات

تسمح نماذج اللغة للروبوتات بـ:
- **فهم** أوامر اللغة الطبيعية
- **تخطيط** تسلسلات المهام
- **الاستدلال** حول الإجراءات

## نماذج VLA

تجمع نماذج الرؤية-اللغة-الفعل بين:
- **الرؤية**: رؤية البيئة
- **اللغة**: فهم الأوامر
- **الفعل**: تنفيذ أوامر المحرك
