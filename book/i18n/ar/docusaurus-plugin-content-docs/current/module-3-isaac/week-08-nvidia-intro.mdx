# الأسبوع 8: مقدمة في NVIDIA Isaac

اكتشف منصة Isaac من NVIDIA للمحاكاة المتقدمة للروبوتات والذكاء الاصطناعي.

## نظام NVIDIA Isaac البيئي

تشمل منصة Isaac:
- **Isaac Sim**: محاكاة واقعية بصريًا
- **Isaac Lab**: إطار تدريب التعلم المعزز
- **Isaac ROS**: حزم ROS 2 المُسرَّعة بـ GPU

## لماذا Isaac؟

- **تسريع GPU** لمحاكاة أسرع
- **عرض واقعي** لمهام الرؤية
- **سير عمل تدريب الذكاء الاصطناعي** المتكامل

## التثبيت

اتبع [دليل تثبيت Isaac Sim](https://docs.omniverse.nvidia.com/isaacsim/latest/installation.html).

## تمرين

قم بتشغيل Isaac Sim واستكشف بيئات العينات.
