# الأسبوع 7: URDF ونماذج الروبوتات

تعلم وصف الروبوتات باستخدام URDF (تنسيق وصف الروبوت الموحد).

## ما هو URDF؟

URDF هو تنسيق XML لتمثيل نماذج الروبوتات، بما في ذلك:
- **الروابط**: الأجسام الصلبة
- **المفاصل**: الاتصالات بين الروابط
- **المرئي**: المظهر
- **التصادم**: الحدود الفيزيائية

```xml
<robot name="my_robot">
  <link name="base_link">
    <visual>
      <geometry>
        <box size="0.6 0.3 0.2"/>
      </geometry>
    </visual>
  </link>
  
  <joint name="wheel_joint" type="continuous">
    <parent link="base_link"/>
    <child link="wheel_link"/>
    <origin xyz="0.2 0 -0.1"/>
    <axis xyz="0 1 0"/>
  </joint>
</robot>
```

## تمرين

أنشئ نموذج URDF لروبوت متنقل بسيط.
