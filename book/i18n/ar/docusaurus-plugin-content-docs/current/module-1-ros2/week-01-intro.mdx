# الأسبوع 1: مقدمة في ROS 2

مرحباً بك في ROS 2! في هذا الفصل، ستتعلم أساسيات نظام تشغيل الروبوت 2.

## ما هو ROS 2؟

ROS 2 (نظام تشغيل الروبوت 2) هو إطار عمل وسيط مفتوح المصدر للروبوتات. يوفر:
- **بنية تحتية للاتصالات** للأنظمة الموزعة
- **أدوات ومكتبات** لبناء تطبيقات الروبوتات
- **تجريد الأجهزة** لمختلف المستشعرات والمشغلات

## المفاهيم الأساسية

### العقد
العقد هي وحدات الحساب الأساسية في ROS 2. كل عقدة تمثل عملية تنفذ مهام محددة.

```python
import rclpy
from rclpy.node import Node

class MyNode(Node):
    def __init__(self):
        super().__init__('my_node')
        self.get_logger().info('تم بدء العقدة!')

def main():
    rclpy.init()
    node = MyNode()
    rclpy.spin(node)
    rclpy.shutdown()
```
