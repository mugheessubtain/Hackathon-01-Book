# Week 9: Isaac Sim for Robot Simulation

Learn to use Isaac Sim for advanced robot simulation scenarios.

## Creating Environments

Isaac Sim provides:
- **Pre-built assets** and robots
- **Physics engines** (PhysX)
- **Sensor simulation** (RGB, depth, lidar)

```python
from omni.isaac.kit import SimulationApp
simulation_app = SimulationApp({"headless": False})

from omni.isaac.core import World
world = World()
world.scene.add_default_ground_plane()
```

## Sensor Integration

- RGB cameras
- Depth cameras
- LiDAR sensors
- Contact sensors
