# Week 8: Introduction to NVIDIA Isaac

Discover NVIDIA's Isaac platform for advanced robot simulation and AI.

## NVIDIA Isaac Ecosystem

The Isaac platform includes:
- **Isaac Sim**: Photorealistic simulation
- **Isaac Lab**: RL training framework
- **Isaac ROS**: GPU-accelerated ROS 2 packages

## Why Isaac?

- **GPU acceleration** for faster simulation
- **Photorealistic rendering** for vision tasks
- **Integrated AI training** workflows

## Installation

Follow the [Isaac Sim installation guide](https://docs.omniverse.nvidia.com/isaacsim/latest/installation.html).

## Exercise

Launch Isaac Sim and explore the sample environments.
