# Week 7: URDF and Robot Models

Learn to describe robots using URDF (Unified Robot Description Format).

## What is URDF?

URDF is an XML format for representing robot models, including:
- **Links**: Rigid bodies
- **Joints**: Connections between links
- **Visual**: Appearance
- **Collision**: Physics boundaries

```xml
<robot name="my_robot">
  <link name="base_link">
    <visual>
      <geometry>
        <box size="0.6 0.3 0.2"/>
      </geometry>
    </visual>
  </link>
  
  <joint name="wheel_joint" type="continuous">
    <parent link="base_link"/>
    <child link="wheel_link"/>
    <origin xyz="0.2 0 -0.1"/>
    <axis xyz="0 1 0"/>
  </joint>
</robot>
```

Create a URDF model for a simple mobile robot.
